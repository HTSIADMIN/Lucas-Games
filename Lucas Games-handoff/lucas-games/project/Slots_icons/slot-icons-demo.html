Drop-in usage

Each icon is a standalone SVG with a transparent background and a 32×32 viewBox. Drop them in /public/slots/ (or wherever) and reference by symbol key. image-rendering: pixelated on the wrapper keeps them crisp at any reel cell size. 

// symbol → asset map
const SYMBOL = {
  boot:    "/slots/icon-boot.svg",
  gun:     "/slots/icon-gun.svg",
  star:    "/slots/icon-star.svg",
  gold:    "/slots/icon-gold.svg",
  sheriff: "/slots/icon-sheriff.svg",
};

// payouts (3oak, 4oak, 5oak) — multipliers of bet per line
const PAYOUT = {
  boot:    [1,  3,   8],
  gun:     [1,  5,  14],
  star:    [2,  8,  25],
  gold:    [4, 15,  60],
  sheriff: [8, 30, 150],
};

// in JSX
<div className="reel-cell">
  <img src={SYMBOL[symbol]} alt={symbol} />
</div>

Add .is-win to a cell for the gold-ring bounce (handy for line wins or "Round 'Em Up" highlight). If you want a SCATTER or WILD distinct from the symbol set, the Sheriff is the strongest candidate — bump it visually with the gold tile background like the paytable's last row.
