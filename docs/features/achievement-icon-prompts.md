# Achievement icon prompts

One prompt per achievement, organized by source. All ~82 achievements
across Penny Pinchers + 12 casino games + the cross-game "meta" set.

## How to use

Each prompt is a single sentence designed for any modern image
generator (Midjourney, DALL-E 3, Stable Diffusion, Imagen, etc.).
Prepend the **shared style preamble** below to every prompt so the
icons land as a coherent set.

### Shared style preamble

Paste this in front of any per-achievement prompt:

```
Pixel art achievement icon, cozy pixel tavern × western × PS1-era
aesthetic, chunky 2-3 pixel outlines, hard drop shadow, no anti-
aliasing, no blur, no rounded corners, vibrant limited palette of
saddle brown / parchment cream / gold / crimson red / cactus green
/ sky blue / ink black, centered subject on a transparent
background, square composition, 128×128 pixels, high contrast,
saloon-poster mood. Subject:
```

So a full prompt for, say, "First Spin" reads:

> *Pixel art achievement icon, cozy pixel tavern × western × PS1-era
> aesthetic, chunky 2-3 pixel outlines, hard drop shadow, no anti-
> aliasing, no blur, no rounded corners, vibrant limited palette of
> saddle brown / parchment cream / gold / crimson red / cactus green
> / sky blue / ink black, centered subject on a transparent
> background, square composition, 128×128 pixels, high contrast,
> saloon-poster mood. Subject: a chunky 3-reel pixel slot machine,
> facing forward, with cherries showing in the center reel and a
> single gold coin sliding out the tray.*

### Rarity treatment (optional accent line)

Append to the prompt if you want a rarity halo per the system's
border-color convention. Most generators understand "glow ring":

| Rarity     | Append                                                             |
|------------|--------------------------------------------------------------------|
| common     | (no halo — base icon only)                                         |
| rare       | `, subtle sky-blue glow ring around the icon`                      |
| epic       | `, soft purple glow ring around the icon`                          |
| legendary  | `, vibrant gold glow ring around the icon, sparkles at the corners`|

The in-app badge renders the border in code, so the halo is just for
the file's first-impression — your call whether to bake it in.

## Generator-specific notes

- **Midjourney / Imagen**: append `--ar 1:1 --style raw` (or your
  generator's equivalent) and consider `--niji 6` if you want a
  slightly more illustrative pixel feel.
- **DALL-E 3**: tends to anti-alias unless you explicitly say "no
  anti-aliasing, hard pixel edges, mosaic of large pixels." Reinforce
  in the prompt if outputs come out smooth.
- **Stable Diffusion / SDXL with pixel-art LoRA**: lower the strength
  on the LoRA to ~0.6 so the western color palette comes through.
- **Batching tip**: render the full set with the same seed + same
  prompt prefix, varying only the per-achievement subject line.
  Keeps the visual style locked across all 82 icons.

## Updating the in-app icon

Once you have an image for an achievement, drop it into
`public/achievements/<source>/<id>.png` and swap the
emoji `icon` field in the catalog (`src/lib/achievements/catalogs/
<source>.ts` or, for PP, the `PP_ICONS` map in
`src/lib/achievements/registry.ts`) for an image path:

```ts
// Before
{ id: "first_spin", icon: "🎰", ... }

// After
{ id: "first_spin", icon: "/achievements/slots/first_spin.png", ... }
```

The `<AchievementBadge>` and `<AchievementToast>` components render
the icon string directly. To make them accept image paths, extend
those two components:

```tsx
{def.icon.startsWith("/") ? (
  <img src={def.icon} alt="" width={32} height={32} />
) : (
  <span aria-hidden>{def.icon}</span>
)}
```

That's a 4-line change in two files. The rest of the system already
threads icons through correctly.
